## **什么是流程图**

使用**图形**表示**思路或想法**的一种极好的方法

#### 怎么表示？

用图形表示，你知道哪些图形？

​		正方形，长方形，三角形，菱形，五角形，六边形等等

> 这些图形代表啥意思？

就有一些约定，大家都遵守的约定

首先需要了解有哪些图形，然后每个图像代表什么意思。

## 画流程图

#### 比较大小

1. 要输入两个数
2. 要把输入的数赋值给两个变量
3. 比较两个数的大小
    1. 如果大于，输出 ... 
    2. 否则， 输出 ... 

![](./res/if2.png)

```python

box = input("请输入整数a: ")
a = int(box)

box = input("请输入整数b: ")
b = int(box)

if a > b:
  print("a is larger")
else:
  print("b is larger")
  
```



#### 计算两个数成绩

1. 输入两个整数
2. 并将两个整数赋值给两个变量，比如a, b
3. 计算 a*b，并将结果赋值给一个变量，不如ab
4. 输出ab的值

![](./res/flow1.png)



```python
box = input("请输入整数a: ")
a = int(box)

box = input("请输入整数b: ")
b = int(box)

ab = a * b

print("a*b结果为："， ab)

```
